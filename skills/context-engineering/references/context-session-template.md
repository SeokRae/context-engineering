# Context Session Template

> Temporary scratch file for maintaining context during multi-step tasks (3 or more steps).
> Used when the execution instruction (Format A) from compose Phase 6 includes a Context Checkpoints block.
> **Must be deleted after task completion.**

## Usage

1. At the start of a task, copy this template and save it as `_context-session.md`
2. After completing each major step, update the relevant section
3. Before starting the next step, re-read the entire file
4. After all steps are complete, delete the file
