단계별 작업 내용

Phase 1 시나리오 분기

A. 그린필드
아무것도 없음
Step 0 대화에서 용어·제약 직접 추출. 최소 Knowledge Base로 Phase 2 진행.
B. 스펙 문서
--specs 문서 있음
병렬 Explore 에이전트로 문서 탐색 → 용어·제약·참조 매니페스트 작성.
C. 기존 코드
코드베이스 있음
코드 스캔 → 기존 패턴·용어·제약 추출. 충돌 가능성 사전 파악.
D. 둘 다
코드 + 스펙
B + C 병렬 수행. 가장 풍부한 Knowledge Base. 불일치 감지 포함.
